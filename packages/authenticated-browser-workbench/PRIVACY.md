# 隐私说明

本模板本身不保存账号凭据，也不向第三方服务上传浏览器 Profile、Cookie 或 Token。实际网页访问由使用者安装的 BrowserSkill、浏览器和目标网站完成，其数据处理方式受各自版本、服务条款与隐私政策约束。

- 只在 `browser-profile.md` 中记录浏览器用途、账号归属类别和禁止使用边界，不写账号名、密码、Cookie、Token、验证码、恢复码或本机实例 ID。
- 站点矩阵只记录“验证了什么、何时验证、有什么限制”，不记录聊天正文、搜索历史、收藏、消息、草稿、后台数据或个人统计。
- 优先新建 BrowserSkill Agent 标签页；只有任务确实依赖现有页面上下文时才借用用户标签页，并在任务结束后归还或停止会话。
- 不读取与当前任务无关的标签页、历史会话、个人资料、消息或通知内容。
- 发布、发送、删除、付款、点赞、评论、关注、上传和提交表单等会改变外部状态的动作，必须在执行前取得明确确认。
- 测试日志、文章素材和公开报告应使用能力层级与故障类型，不应包含账号标识、页面私密内容或后台数字。

登录态意味着页面可能暴露敏感信息。即使工具可以读取，也不代表任务有权读取、保存或公开这些信息。
