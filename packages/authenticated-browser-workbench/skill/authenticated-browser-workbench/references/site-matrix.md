## 状态口径

- `已验证`：代表性真实任务完成目标业务结果，并完成会话清理。
- `受限`：只完成部分阶段，或结果质量、来源、速度、账号权限仍有缺口。
- `失败`：当前路径无法完成任务，已停止并保留可解释证据。
- `未测`：没有当前证据，不依据产品宣传或登录成功推断能力。

## 维护规则

1. 只记录能够长期复用的路由、结果层级、限制和测试日期。
2. 不记录账号标识、聊天内容、搜索历史、后台数据、临时实例 ID 或元素编号。
3. 页面改版、账号变化或新实测与旧结论冲突时，以当前事实替换旧结论。
4. 同类站点不要求全部测试。候选期优先覆盖连接、登录态复用、只读检索、实例选择和会话停止。
