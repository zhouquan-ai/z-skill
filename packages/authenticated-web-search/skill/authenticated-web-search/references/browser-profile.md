## 账号硬边界

- 列出哪些浏览器中的账号属于本人、家人、单位或测试环境。
- 对非本人账号默认禁止打开、借用、读取或操作，除非账号所有人对当前具体任务明确授权。
- 不在本文件写账号名、邮箱、手机号、密码、Cookie、Token、验证码或恢复码。
- 不把“浏览器已经登录”解释为“当前任务有权读取全部内容”。

## 默认确认边界

站内搜索、只读翻页和读取当前任务结果可以直接执行。发布、发送、删除、付款、上传、点赞、评论、关注、修改账号和提交会改变外部状态的表单，必须在动作前取得明确确认。
